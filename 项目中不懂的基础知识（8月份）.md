### 1. Filter()
> filter()方法利用所有通过指定函数测试的元素创建一个新的数组，并返回。

> 注意：两个数组间的对比要用id,不用索引

#### 如何获取对应的id

栗子：

```js
var arr = [{
	id : 1,
	name : 'use'
}, {
	id : 2,
	name : 'you zi'
}];

var getArrId = function(id) {
	return arr.filter(function(objArr) {
		return +objArr.id === id;
	});
};

getArrId(2); //[{id: 2,name: "you zi"}]
```
